# TodoApp

***
## Приложение:
Это список дел для того чтобы контролировать и не забывать дела, которые нужно сделать за день.
***
## Стек:
  - React.js
  - Redux/toolkit
  - RTK-query
  - MySql
  - Express
  - SCSS
***
## Скрипты:
1. Для скачивания зависимостей:
```
npm install
```
2. Для запуска в режиме разработки:
```
npm run dev
```
2. Для сборки frontend:
```
npm run build
```
